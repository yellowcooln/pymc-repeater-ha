# pyMC Repeater Home Assistant Add-on Repository

This repository contains a Home Assistant add-on for running
[`pyMC_Repeater`](https://github.com/pyMC-dev/pyMC_Repeater).

## Add-ons

### [`pymc_repeater`](./pymc_repeater)

Runs pyMC Repeater inside Home Assistant with persistent config and data
storage.
